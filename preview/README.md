# Preview Assets

Files in this directory are for local theme development and review.

- `index.html` renders a Markdown-like document in the browser.
- `markdown.css` styles the browser preview page.
- `cursor-preview.css` styles Cursor's side Markdown preview (`Ctrl+K V`).

Use these files when you want to iterate on layout, typography, spacing, and generic Markdown rendering before packaging a dedicated inline theme.
