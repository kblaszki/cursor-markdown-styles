# Lumina — historia wersji

Sygnaturowy motyw inline preview z gradientami i kolorowymi elementami.

## Pliki

| Plik | Wersja | Opis |
|------|--------|------|
| `cursor-inline-lumina-v1-neon.css` | v1 | Pierwsza wersja — jaskrawa paleta, mocne glow |
| `cursor-inline-lumina-v2-muted.css` | v2 | Stonowana — przygaszone kolory, subtelne cienie |
| `cursor-inline-lumina-v3-balanced.css` | v3 | Zbalansowana — srodek miedzy v1 a v2 **(rekomendowana)** |

## Skrot

`../cursor-inline-lumina.css` to kopia **v3** — wygodny domyslny plik do kopiowania do Cursora.

## Instalacja

Skopiuj wybrana wersje do:

```
C:\Program Files\cursor\resources\app\out\vs\workbench\cursor-inline-preview.css
```

Zrestartuj Cursor.

## Zasada na przyszlosc

Kazda kolejna iteracja Lumina = nowy plik z kolejnym numerem (v4, v5...).
Nie nadpisuj starych wersji.
