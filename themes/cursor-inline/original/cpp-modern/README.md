# C++ Modern

A Cursor inline preview theme family inspired by VS Code's modern dark C++ feel.

## Files

- `cursor-inline-cpp-modern.css` - the recommended installable version
- `cursor-inline-cpp-modern-v1-syntax.css` - an earlier syntax-driven variant
- `cursor-inline-cpp-modern-v2-readable.css` - the historical readable variant kept for comparison

## Notes

- This directory exposes one recommended public install file and keeps older iterations for comparison.
- The current theme focuses on readability with a restrained blue and teal palette.
- For VS Code / Cursor **side preview** (`markdown.styles`), use the ported equivalents in [themes/vscode-preview/original/cpp-modern/](../../vscode-preview/original/cpp-modern/).

## Install

Copy the full contents of the chosen file into:

```text
C:\Program Files\cursor\resources\app\out\vs\workbench\workbench.desktop.main.css
```

Per-user install:

```text
%LOCALAPPDATA%\Programs\cursor\resources\app\out\vs\workbench\workbench.desktop.main.css
```

Recommended source file:

```text
themes/cursor-inline/original/cpp-modern/cursor-inline-cpp-modern.css
```
