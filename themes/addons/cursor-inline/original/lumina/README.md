# Lumina

A signature Cursor inline preview theme family built around luminous gradients and bold blue accents.

## Files

- `cursor-inline-lumina.css` - the recommended installable version
- `cursor-inline-lumina-v1-neon.css` - the original vivid neon version
- `cursor-inline-lumina-v2-muted.css` - a softer, lower-glow variant
- `cursor-inline-lumina-v3-balanced.css` - a midpoint between vivid and muted
- `cursor-inline-lumina-v4-neon-blue.css` - the historical blue-focused version kept for comparison

## Notes

- This directory exposes one recommended public install file and keeps older iterations for comparison.
- The current Lumina theme uses a unified blue neon palette.
- For daily Markdown work in VS Code or Cursor, prefer the MPE setup in [themes/mpe/](../../../../mpe/).
- The MPE equivalents are now available as `themes/mpe/global-lumina.less` and `themes/mpe/global-lumina-light.less`.

## Install

Copy the full contents of the chosen file into:

```text
C:\Program Files\cursor\resources\app\out\vs\workbench\workbench.desktop.main.css
```

Per-user install:

```text
%LOCALAPPDATA%\Programs\cursor\resources\app\out\vs\workbench\workbench.desktop.main.css
```

Recommended source file:

```text
themes/addons/cursor-inline/original/lumina/cursor-inline-lumina.css
```
