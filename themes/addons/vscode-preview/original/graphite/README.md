# Graphite

An original documentation-first theme family focused on long-form readability, quiet surfaces, and warm copper accents.

## Files

- `vscode-preview-graphite.css` - dark editorial variant
- `vscode-preview-graphite-light.css` - light editorial variant

## MPE Bundles

- `themes/mpe/global-graphite.less`
- `themes/mpe/global-graphite-light.less`

## Notes

- Graphite is designed for technical documentation, notes, and prose-heavy Markdown.
- Compared with `cpp-modern`, it uses lower-saturation accents and calmer code/table surfaces.
