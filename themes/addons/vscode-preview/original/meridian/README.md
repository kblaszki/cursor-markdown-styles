# Meridian

An original navigation-oriented theme family with blue-green accents, crisp separators, and strong structure for tables and checklists.

## Files

- `vscode-preview-meridian.css` - dark variant
- `vscode-preview-meridian-light.css` - light variant

## MPE Bundles

- `themes/mpe/experimental/meridian/style.less`
- `themes/mpe/experimental/meridian-light/style.less`

## Notes

- Meridian is meant for structured documentation, runbooks, and checklists.
- Compared with Lumina, it is calmer and more hierarchical, with less glow and tighter table contrast.
