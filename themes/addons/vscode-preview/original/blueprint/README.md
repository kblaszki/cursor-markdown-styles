# Blueprint

Architectural cyanotype family: drafting ink, dashed rules, double-line quotes, and faint blueprint grid lines in code fences.

## Files

- `vscode-preview-blueprint.css` - deep cyanotype board
- `vscode-preview-blueprint-light.css` - blue ink on vellum

## MPE Bundles

- `themes/mpe/global-blueprint.less`
- `themes/mpe/global-blueprint-light.less`
