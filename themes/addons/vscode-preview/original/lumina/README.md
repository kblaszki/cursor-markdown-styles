# Lumina

An original luminous blue theme family with gradient headings, bright links, and higher-energy surfaces than the other MPE bundles.

## Files

- `vscode-preview-lumina.css` - dark neon variant
- `vscode-preview-lumina-light.css` - light icy variant

## MPE Bundles

- `themes/mpe/global-lumina.less`
- `themes/mpe/global-lumina-light.less`

## Notes

- Lumina keeps its expressive blue identity in both modes.
- The light variant is tuned manually for readability instead of being an automatic inversion of the dark palette.
