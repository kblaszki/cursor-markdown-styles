# Phosphor

CRT terminal family. Headings lean monospace, lists use terminal bullets, and fenced blocks get faint scanlines.

## Files

- `vscode-preview-phosphor.css` - P1 green phosphor
- `vscode-preview-phosphor-amber.css` - P3 amber phosphor (paired as the “light” sibling)

## MPE Bundles

- `themes/mpe/experimental/phosphor/style.less`
- `themes/mpe/experimental/phosphor-amber/style.less`
