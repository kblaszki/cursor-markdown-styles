# Matcha

Tea-garden documentation family: soft serif prose, moss + gold accents, rounded code panels, and a calm lacquer night / steamed paper pair.

## Files

- `vscode-preview-matcha.css` - lacquer night
- `vscode-preview-matcha-light.css` - steamed paper

## MPE Bundles

- `themes/mpe/experimental/matcha/style.less`
- `themes/mpe/experimental/matcha-light/style.less`
